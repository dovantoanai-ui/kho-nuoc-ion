# Kho Nước Ion Kiềm — DA05

App quản lý kho nhập–xuất–tồn + công nợ cho nước ion kiềm (nhà phân phối). Chạy trên trình duyệt điện thoại + máy tính, đồng bộ Google Sheet.

- `index.html` — app (GitHub Pages phục vụ ở gốc).
- `backend-apps-script.gs` — code dán vào Apps Script của Google Sheet kho.
- `.nojekyll` — để GitHub Pages phục vụ file tĩnh nguyên trạng.

## Link web (sau khi bật Pages)
`https://dovantoanai-ui.github.io/kho-nuoc-ion/`

## Cách chạy
Xem hướng dẫn cài đặt trong app (tab Danh mục) và file backend. Cần deploy Apps Script Web App rồi dán URL `/exec` vào app.
